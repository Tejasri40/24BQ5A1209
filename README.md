## Setup

Install dependencies

```bash
npm install
```

Run application

```bash
npm start
```

Server starts on:

```text
http://localhost:3000
```

## API Endpoint

```text
GET /top-notifications
```

## Priority Rules

```text
Placement > Result > Event
```

If two notifications have the same type, the newer notification receives higher priority.

## Output

Returns the top 10 highest priority notifications.
